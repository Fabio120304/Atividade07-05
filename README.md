# 🎬 Sistema de Rede de Cinemas

## 📖 Descrição do Projeto

Este projeto consiste no desenvolvimento de um sistema de informação para gerenciamento de uma rede de cinemas, permitindo o controle de filmes em cartaz, sessões, registro de público e consultas de informações relacionadas aos cinemas e filmes.

O sistema foi desenvolvido utilizando arquitetura em camadas com padrão MVC, Service e Repository, além de persistência de dados em SQLite.

---

# 🚀 Tecnologias Utilizadas

- Python
- SQLite
- VS Code
- MVC Architecture
- Service Layer
- Repository Pattern
